# Network Packet Analyzer

## About
A Python-based tool for analyzing network packets.

## Features
- Capture packets
- Display source IP
- Display destination IP
- Identify protocols

## Technologies
- Python
- Scapy

## Author

```bash
pip install -r requirements.txt
```

```bash
python packet_analyzer.py
